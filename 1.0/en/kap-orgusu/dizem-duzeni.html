--- layout: s1n0/default title: Rhythm Layout - Grid System ---

A rhythmic layout has been developed to create repetitive scopes of the same width. If you want to create one-one, two-two, three-three, ... you just need to use the parameter from 1r to 24r. "r" mean is "repeat".
