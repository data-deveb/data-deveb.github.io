Varsayılan olarak bazı biçimlerde tasarımlarınızı geliştirirsiniz. Yeri geldiğinde farklı biçimleri kullanırsınız. İşte bu bölümde bulunan biçim çeşitleri işlerinizi kolaylaştıracaktır.

Tam Ekran

Tasarımlarınızı tam ekran düzeninde yapmak istiyorsanız deveb tarafından hazırlanan "tamekran" değiştirgesini kullanacaksınız demektir. HTML body etiketine istediğiniz ekran aralığını belirterek kullanabilirsiniz.

Örnek

Bu örneğin canlı durumunu görmek için tıklayın.

{% highlight html %} Bu alanda bulunan her şey tam ekran düzeni içerisinde kalır. {% endhighlight %}

Kap

Tasarımlarınızı yaparken ana genişliği sabit genişlik olan tasarımlar yapın. Aynı zamanda bu genişlikler ekran genişliğine göre uyarlanmıl sabit genişliklerde olsun. Bunun için "kap" değiştirgesini kullanmanız yeterlidir.

Örnek

1 3p
1 3p
1 3p
{% highlight html %}
1 3p
1 3p
1 3p
{% endhighlight %}

Sığ

Tasarımlarınızda yeri geldiğinde bir alana enine boyuna bir alan sığdırmak isteyebilirsiniz. O zaman geldiğinde ekran örgüsü içerisinde "sığ" değiştirgesini kullanmanız yeterli olacaktır. Böylelikle alan bulunduğu alana sığmış durumda olacaktır. Bazı durumlarda "sütun" değiştirgesiyle birlikte kullanmanız gerekebilir. Daha iyi anlamak için örnekleri inceleyiniz.

Örnek

Bu örneğin canlı durumunu görmek için tıklayın.

{% highlight html %}
Bu alan tam ekrana göre sığdırılmıştır.
{% endhighlight %}

0 (sıfır)

Tasarımlarınızı yaparken içeriğe göre genişlik kullanmak isteyebilrsiniz. Bunun için "0" değiştirgesini kullanmanız yeterlidir.

Örnek

0
1 3p
1 3p
{% highlight html %}
0
1 3p
1 3p
{% endhighlight %}
