Yalnız Ekran Alone Screen bunu-sınıf class-it Sadece tanımlandığı ekran grubuna özellik atar Only assigns a property to the screen group that you define
Diğer Ekranlar Other Screens diğerleri-sınıf class-others Tanımlandığı ekran grubundakiler dışında kalanlara özellik atar Assigns properties to those outside the defined screen group
Büyük Ekranlar Bigger Screens büyükleri-sınıf class-bigger Tanımlandığı ekrandan daha büyük ekranlara özellik atar Assigns features to screens bigger than the defined screen
Küçük Ekranlar Smaller Screens küçükleri-sınıf class-smaller Tanımlandığı ekrandan daha küçük ekranlara özellik atar Assigns features to screens smaller than the defined screen
